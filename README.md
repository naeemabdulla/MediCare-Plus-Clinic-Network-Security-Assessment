## 📌 Project Overview
This repository contains all project artefacts for the **Network Security Assessment of MediCare Plus Clinic**, completed as part of **COIT13236 — Capstone Project** at CQUniversity.

Our project focuses on evaluating the clinic’s **security posture**, identifying network vulnerabilities, and recommending mitigation strategies to improve overall cyber resilience.

## 🎯 Project Objectives
- Assess MediCare Plus Clinic’s current **network security posture**
- Identify **vulnerabilities, threats, and associated risks**
- Conduct **stakeholder interviews** to gather operational requirements
- Review and evaluate **cloud and system architecture**
- Provide **actionable security recommendations**
- Develop a **risk register** aligned with industry standards
